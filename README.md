# Poker Timer

A fully featured home game poker tournament manager that runs as a single HTML file — no install, no build step, works on any device.

## Features

- **Timer** — blind level countdown, break timer, turbo mode, auto-advance
- **Players** — add players by name, track eliminations and rebuys, prize pool auto-updates
- **Blinds** — Turbo / Standard / Deep Stack templates, fully editable schedule, mid-game turbo mode
- **Chips** — custom chip denominations and colors for your specific set
- **Prizes** — auto-scales payouts based on player count, or set custom percentage splits
- **Themes** — dark and light mode

## Usage

Open `index.html` in any browser. No dependencies, no server required.

Hosted at: https://pmontalvo.github.io/Poker-Timer
